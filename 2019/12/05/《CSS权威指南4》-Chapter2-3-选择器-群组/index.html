分组

这个声明相当于一个分组选择器，它列出文档中包含的每个元素，通用选择器只需敲击键盘一次，就能把文档中所有元素的 color 属性赋值为 red。 但是，要注意：虽然通用选择器很方便，但是它的特性是 0-0-0；由于它的目标是其声明范围内的所有内容，因此它可能产生意想不到的结果，本书稍后对此进行讨论。
